# Email: Export/Import

## Meta
- **Subject Line:** Your AI memories can now move with you
- **Preview Text:** Export, import, and merge — with smart deduplication
- **Audience Segment:** All users

## Body

### Opening
Starting today, your AI memory is portable. Export knowledge from any Memories instance, import it into another — and the system handles duplicates automatically.

### What's New
- **Selective export** — Choose what to transfer: all memories, one project, or a specific date range
- **Smart import** — Three strategies from fast bulk insert to AI-powered conflict resolution
- **Automatic safety** — Every import creates a backup first. One command to restore if needed.

### How to Get Started
```bash
memories export -o backup.jsonl
memories import backup.jsonl --strategy smart
```

### Closing
Your knowledge compounds faster when it's not trapped in one place.
