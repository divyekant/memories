# Email Announcement: CLI

## Subject: Your AI memories are now one command away

## Body

Every memory operation your server supports is now a terminal command.

The Memories CLI gives you and your AI agents full access to 30+ operations -- search, add, batch import, delete, backup, sync, extract -- without writing HTTP code or opening a browser.

**How it works:** The CLI detects its environment automatically. You get colored, readable output when typing interactively. AI agents get structured JSON when they pipe. No flags needed.

**What you can do:**
- Search memories instantly: `memories search "deployment patterns"`
- Batch import from JSONL files at scale
- Chain commands with pipes and branch on exit codes
- See exactly where each config value comes from with `memories config show`

**Get started in seconds:**

```
pip install memories
memories admin health
```

No Docker needed for CLI usage. No configuration required. Works immediately against your running Memories server.
