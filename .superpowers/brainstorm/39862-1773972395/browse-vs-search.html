Key Differences

Browse Mode
  • No left borders — clean cards
  • No score badges
  • No feedback buttons
  • Shows timestamp + source
  • Archived items shown at 50% opacity with badge
  • Pin badge visible
Search Mode
  • Small dot instead of thick left border (green/yellow/red)
  • Score badge with percentage (color-coded)
  • ▲/▼ feedback buttons
  • Query term highlighting in gold
  • Score tooltip on hover (explain breakdown)
  • Archived items excluded by default
